# 🚀 Preparation Tracker

## 📖 Overview

Preparation Tracker is a React.js application that helps students organize and manage their placement preparation in one place. It provides a clean dashboard to track DSA problems, learning goals, interview experiences, and overall progress.

---

## ✨ Features

- 📚 Track DSA and coding problems
- 🎯 Create and manage preparation goals
- 💼 Store interview experiences and feedback
- 📊 Interactive dashboard
- 🌙 Dark & Light theme
- 📱 Responsive design
- 💾 Import & Export JSON data

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- HTML5
- CSS3

---

## 🎯 Project Purpose

The goal of this project is to help students stay organized throughout their placement journey by keeping coding practice, interview preparation, and learning goals in one dashboard.
